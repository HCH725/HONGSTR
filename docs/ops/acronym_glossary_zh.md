# HONGSTR Acronym Glossary (zh-TW)

- SSOT: 單一真實來源（Single Source of Truth）
- DD: 回撤（Drawdown）
- MDD: 最大回撤（Max Drawdown）
- Sharpe: 風險調整後報酬指標（Sharpe Ratio）
- Trades: 交易筆數
- OOS: 樣本外（Out-of-Sample）
- IS: 樣本內（In-Sample）
- OOS/IS: 樣本外 / 樣本內（Out-of-Sample / In-Sample）
- WF: 滾動前推驗證（Walk-Forward）
- TP: 停利（Take Profit）
- SL: 停損（Stop Loss）
- DCA: 定期定額 / 分批攤平（Dollar-Cost Averaging）
- L1: 低優先級處置
- L2: 中優先級處置
- L3: 高優先級處置
- L1/L2/L3: 低 / 中 / 高優先級風險分層
