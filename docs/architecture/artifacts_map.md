# HONGSTR Artifacts Map

This document serves as the SSOT (Single Source of Truth) for locating key system artifacts.

## 1. Research & Backtesting Artifacts

Located under `data/backtests/YYYY-MM-DD/[BT_ID]/`.

- **`summary.json`**: High-level PnL, Sharpe, and trade statistics.
- **`selection.json`**: Technical metadata for strategy selection (parameter space).
- **`gate.json`**: Compliance status (G0-G6) and DoD checklist results.

## 2. Global State Artifacts

Located under `data/state/`.

- **`system_health_latest.json`**: The ultimate SSOT health indicator (Freshness, Coverage, Brakes).
- **`freshness_table.json`**: P50/P95 latency statistics for all data symbols.
- **`coverage_matrix_latest.json`**: Tracking of backtest data gaps and aggregations.
- **`brake_health_latest.json`**: State of safety brakes and regime monitors.

## 3. Strategy Pool & Ranking

Located under `data/state/` and `data/state/_research/`.

- **`strategy_pool.json`**: The active shelf of strategies (Candidates/Promoted/Demoted).
- **`strategy_pool_summary.json`**: Human-readable summary of the current pool state.
- **`_research/leaderboard.json`**: Periodic ranking of potential candidates from research.

## 4. Derived Data

Located under `data/derived/[SYMBOL]/[TIMEFRAME]/`.

- **`klines.jsonl`**: The primary backtest data source (ingested from Binance).
- **`features/*.parquet`**: Persistence layer for ML/Technical features.
